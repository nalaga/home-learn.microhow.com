도쿄 4박5일 - 2일차 일정 요약

도쿄 4박5일 : 2일차 일정 요약

2016.08.20~24

숙소 출발 → 24시간 소바&우동 → 후지사와 → 에노덴 관광(에노시마, 가마쿠라고교, 하세, 가마쿠라)

→ 요코하마(SEA BASS, 아카렌가 창고, 코스모월드) → 회전초밥집 → 숙소

에노덴

에노시마 가마쿠라고교 아카렌가 창고

알차게 보냈던 2일차!

도쿄에 간다면 하루쯤은 근처인 요코하마와 에노시마 관광을 하면 좋을 듯

(사실 하루도 너무 부족함 )

- 오전(?)에 기상해서 준비 단단히 하고 나옴 (멀리가니까)

- 닌교초역 사거리에 있는 24시간 소바&우동집에서 냠냠 (싼데 맛있음)

- 지하철을 타고 후지사와로 이동(아사쿠사선 타고 신바시에서 JR도카이도본선으로 갈아타고 쭉)

- 후지사와역에서 에노덴 1일 자유이용권 삼

- 맞은편에서 소다 한 잔 마시고 에노덴 탑승 (에노덴♥)

- 에노시마역에서 내림

- 에노시마까지 엄청 걸어감. 근데 재밌음. 후지산도 봤음. 짱 더워서 우산쓰고 걸어감

- 운세 한 번 뽑고 에스컬레이터 이용권사서 타고 정상까지 감

- 타코(문어)&에비(새우) 센베이랑 만두랑 맥주 먹음 (땀 흘리고 먹어서 핵꿀맛)

- 코스프레 하는 친구들도 많이 봄 (여기가 코스프레 핫플레이스인가봐)

- 전망대 올라갔다가 시라스동 찾아 내려옴

- 아무데나 들어갔다가 살아있는 시라스동 못먹고 죽은 시라스동이랑 자루소바 먹음 (근데 맛있다..)

- 내려오는 길을 잃고 이상한 동굴앞까지 감(이미 예상했던 시간이 많이 지남)

- 계속 내려오면서 아이스크림이랑 나마시라스(살아있는 거) 500엔인가 파는거 사먹음

- 다시 에노시마역으로 돌아와서 (이미 지침) 에노덴타고 가마쿠라고교에서 내림

- 강백호마냥 사진하나 찍고 바로 다시 에노덴 탑승

- 하세 역에서 내려서 불상보러 갔는데, 이미 문닫은 시간이라 못 봄 (근데 너무 커서 위로 다 보임)

- 다시 에노덴타고 가마쿠라 역에 갔다가 지하철 타고 요코하마로 바로 이동

- 요코하마에서 미나토미라이선 타고 모토마치로 이동

- 모토마치에서 SEA BASS 아카렌가까지 가는거 끊어서 배에서 야경 구경 (이거 좋음)

- 내려서 사진찍고 가는데, 아라켄가 창고에서 축제 중

- 칵테일 한 잔씩 먹고, 길거리 농구도 하길래 구경하다가 코스모 월드까지 걸어감 (20분)

- 코스모월드 맞은 편에 건물 1층에서 회전초밥집 있길래 거기서 회전초밥 먹음

- 닌교초로 돌아와서 컵라면이랑 야식사들구 숙소와서 냠냠

- 발바닥에 휴족시간 붙이고 잤나? 암튼 잠

- 끝 -
