오사카 3박4일 - 1일차 일정 요약

< 오사카 3박4일 : 1일차 일정 요약 >

인천공항 출발 → 간사이공항 → 숙소(나니와호텔) → 스시잔마이 → 오사카성 → 한큐백화점

→ HEP FIVE 관람차 → 우메다 공중정원 → 가츠동집 → 꼬치집(이자카야) → 숙소

스시잔마이 오사카성 HEP FIVE 관람차 우메다 스카이빌딩

- 인천공항에서 10시 15분 peach 항공기로 출발 (약간의 연착이 있어 30분쯤 출발)

- 12시 10분 간사이공항 도착 후 입국수속

- 간사이 공항에서 [오사카주유패스 난카이선] 2,900엔 구입

- 급행열차타고 난바역 하차 후 숙소[나니와호텔]까지 도보로 이동 (20분)

- 체크인 후 짐만 맡기고 2시 반쯤 출발

- 도톤보리 [스시잔마이]에서 점심으로 초밥 (굿)

- 난바역 가는 길에 홉슈크림 커스타드맛, 녹차맛 (굿)

- 다니마치욘초메 역으로 이동하여 오사카성 구경 (주유패스 사용, 내부는 박물관 같고 별로 재미없음)

- 녹차아이스크림 (굿)

- 오사카성 입구에서 앉아서 엄청 많은 학생들 구경 (물어봤더니 코러스 연습하러 왔다고 함)

- 히가시우메다 역으로 이동

- 한큐백화점에서 도지마롤 구입

- 7층 HEP FIVE 관람차 탑승 (주유패스 사용, 무서웠는데 좋았다. 꼭 타보길!)

- 8층 HEP HALL 에서 오락실 구경 및 뽑기

- 우메다 스카이빌딩으로 이동 (지하도로가 사람은 많은데 으스스하게 생김)

- 우메다 공중정원 구경 (주유패스 사용, 야경이 예쁘다)

- 지하 1층 타키미코지에서 저녁으로 오코노미야키 먹으려다 줄이 너무 길어서 가츠동 먹음 (굿)

- 닛폰바시 역으로 이동해 숙소에서 짐 풀고 옷 갈아입음

- 도톤보리 꼬치집(이자카야)에서 하루 마무리

- 편의점에서 호로요이, 과자, 우유 등 주전부리 구입

- 숙소로 이동 후 꿀잠 (돌아다녀서 발바닥 겁나 아픔ㅠㅠ)

- 끝 -
